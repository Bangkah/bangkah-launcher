# Contributing Guide

Terima kasih ingin berkontribusi! Ikuti langkah berikut:

1. Fork repo ini dan buat branch baru untuk fitur/bugfix Anda.
2. Pastikan code sudah di-test dan lulus CI.
3. Sertakan deskripsi jelas pada PR.
4. Ikuti code style dan best practice Laravel.
5. Untuk diskusi fitur/ide, gunakan GitHub Discussions.

Kontributor baru sangat dihargai!
