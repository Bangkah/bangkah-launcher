# Bangkah v1.0.5 Release Notes

Release Date: March 15, 2026

Focus: Launcher recovery, repository cleanup, and test stability.

## Highlights

- Restored Bangkah launcher command registration so `bangkah:create` is available again.
- Fixed launcher integration path in the project so command help and namespace discovery work reliably.
- Stabilized feature testing by removing strict dependency on built Vite assets during automated tests.
- Cleaned repository structure by removing duplicate sandbox project copies previously tracked in Git.
- Added repository hygiene rules in `.gitignore` and documented expected layout in `docs/PROJECT_STRUCTURE.md`.

## Validation Summary

- `php artisan list bangkah` shows `bangkah:create`.
- `php artisan bangkah:create --help` runs successfully.
- `php artisan test` passes.

## Notes

This release focuses on reliability and maintainability. No breaking API changes were introduced for end users.
