# Bangkah Public Roadmap

## Visi Jangka Panjang
Bangkah bertujuan menjadi starter kit Laravel paling praktis, aman, dan siap produksi untuk developer Indonesia dan global. Kami ingin setiap project baru bisa dimulai dengan best practice, minim setup manual, dan siap scale.

## Roadmap 6–12 Bulan ke Depan

### Q1–Q2 2026
- [ ] Integrasi template frontend tambahan (React, Vue, Svelte)
- [ ] Opsi deployment ke cloud (Vercel, Railway, DigitalOcean)
- [ ] CLI audit: validasi template agar tidak ada malicious code
- [ ] Dokumentasi E2E test & CI/CD lebih lengkap
- [ ] Fitur auto-update dependency (integrasi Dependabot lebih dalam)

### Q3–Q4 2026
- [ ] Marketplace template komunitas (dengan review/security check)
- [ ] Integrasi monitoring/logging siap pakai (Sentry, Bugsnag, dsb)
- [ ] Fitur rollback/scaffold undo
- [ ] Dukungan multi-bahasa (EN/ID)
- [ ] Kolaborasi dengan starter kit open source lain
